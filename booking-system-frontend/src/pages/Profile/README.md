# 个人中心页面

## 功能概述

个人中心页面提供了完整的用户账户管理功能，包括：

1. **个人信息查看**
   - 用户名
   - 姓名
   - 身份证号
   - 手机号
   - 邮箱
   - 账户余额
   - 用户角色
   - 注册时间

2. **余额管理**
   - 查看当前余额
   - 在线充值功能（支持10-10000元范围）
   - 预设充值金额：50、100、200、500、1000、2000元
   - 自定义充值金额输入
   - 余额变动记录查询
   - 记录类型：充值、支付、退款、改签

3. **修改密码**
   - 输入原密码验证
   - 设置新密码（6-20位，需包含字母和数字）
   - 确认新密码
   - 修改成功后自动跳转登录页

## 使用方式

### 访问路径
- 路由：`/profile`
- 需要登录认证才能访问

### 入口
- 点击页面右上角头像
- 在下拉菜单中选择"个人中心"

## 技术实现

### 状态管理
- 使用 Redux 管理用户信息状态
- 使用 `getUserProfile` action 获取最新用户数据

### API 接口
- `GET /user/profile` - 获取用户信息
- `PUT /user/password` - 修改密码
- `POST /user/balance/recharge` - 充值余额
- `GET /user/balance/history` - 获取余额变动历史

### 组件特性
- 使用 Ant Design Tabs 组件实现多标签页切换
- 表单验证确保数据有效性
- 加载状态提示
- 错误处理和用户反馈

## 样式特点

- 响应式设计，适配移动端和桌面端
- 余额卡片使用渐变背景突出显示
- 表格自适应滚动
- 统一的视觉风格

## 注意事项

1. **充值限制**
   - 单次充值金额：10-10000元
   - 充值后余额即时到账

2. **密码要求**
   - 6-20位字符
   - 必须包含字母和数字
   - 修改成功后需要重新登录

3. **权限控制**
   - 必须登录后才能访问
   - 通过 PrivateRoute 组件保护路由
